> The Command Line - Simplified

What is the command line?

The command line is a user interface to used to navigate and interact with the computer

  • The command line accepts prompts or commands from the user
  • On mac and linux, the command line is the same as the terminal

What does it do?

It is very similar to a GUI (graphical user interface), like the file explorer, except instead of using your mouse to perform actions, you use commands

  • This allows for quicker access, more efficiency, and more elaborate commands to be performed

Commands to know

  • cd
  • ls/dir
  • clear/cls
  • touch/echo
  • mkdir
  • mv
  • nano
  • PATH
  • pwd
  • rm
