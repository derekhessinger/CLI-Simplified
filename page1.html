> More Commands to Learn

Congratulations, you made it!

You've succesfully used the three basic navigation commands to get to another page. Now, we're going to learn how to manipulate files in the directory.

Directory Manipulation

Here are a few more commands that you'll likely encounter pretty often while using the terminal. These are similar to cd because they need you to specify a target path or destination:

  • mkdir x - creates a new directory x within the one you are currently in
  • rmdir y - remove the specificied directory y, but only if y is empty

File Manipulation

Apart from manipulating directories, you might also need to work with files:

  • touch x - creates a new emtpy file inside the current directory called x
  • rm y - deletes the specificied file y

Try it out!

Make a new empty directory and file (name them whatever you'd like) and then remove both!
